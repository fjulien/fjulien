## Les variables

article
Les variables en JavaScript sont un outil fondamental pour stocker des valeurs dans votre code. Elles vous permettent de donner un nom à une valeur spécifique, de sorte que vous pouvez référencer cette valeur plus tard dans votre code. Il existe trois façons de déclarer des variables en `var, let et const`.

La déclaration de variable la plus courante est var. Elle permet de déclarer une variable qui peut être réaffectée à une nouvelle valeur plus tard dans le code. Par exemple:

```javascript
var maVariable;
maVariable = "Ceci est une chaîne de caractères";
maVariable = 42; // la variable peut être réaffectée à une nouvelle valeur
```

```javascript
var maVariable;
maVariable = "Ceci est une chaîne de caractères";
maVariable = 42; // la variable peut être réaffectée à une nouvelle valeur
```

Il est à noter que les variables déclarées avec var ont une portée de fonction, c'est-à-dire qu'elles sont accessibles dans toute la fonction ou dans le script où elles ont été déclarées, même si elles ont été déclarées à l'intérieur d'un bloc.

Depuis ECMAScript 6, il est possible de déclarer des variables avec let et const. let permet de déclarer une variable qui peut être réaffectée à une nouvelle valeur plus tard dans le code, tandis que const déclare une variable qui ne peut pas être réaffectée.

```javascript
let maVariable2 = "Ceci est une autre chaîne de caractères";
maVariable2 = "Nouvelle valeur"; // la variable peut être réaffectée

const maVariable3 = "Ceci est une chaîne de caractères";
maVariable3 = "Nouvelle valeur"; // cela va générer une erreur
```

Il est important de noter que les variables déclarées avec let et const ont une portée de bloc, c'est-à-dire qu'elles sont uniquement accessibles dans le bloc où elles ont été déclarées.

Il existe plusieurs types de données que vous pouvez stocker dans des variables, tels que des chaînes de caractères, des nombres, des booléens, des tableaux et des objets. Il est important de choisir des noms de variables descriptifs pour faciliter la compréhension de votre code. Il est également recommandé d'utiliser let et const à la place de var car ils ont un comportement plus prévisible et sont plus sûrs à utiliser.

  Les variables sont des symboles qui associent un nom (l'identifiant) à une valeur. Les variables peuvent changer de valeur au cours du temps (dynamique) ou au contraire être figées (constante).

Plusieurs élements caratérise une variable:
- Son nom
- Son type
- Sa valeure
- Sa protée
- Sa visibilitée
- Sa durée de vie

  ### Les bases
  #### VAR
La déclaration <bold><abbr title="pour variable">var</abbr></bold> permet de
    déclarer une variable et éventuellement d'initialiser sa valeur.
  #### LET

    La déclaration <bold>let</bold> permet de déclarer une variable dont la portée
    est celle du bloc courant.

  #### CONST

    La déclaration <bold>const</bold> permet de créer une constante accessible uniquement
    en lecture. La valeur d'une constante ne peut pas être modifiée par des réaffectations
    ultérieuresou ou être re-déclarée. Par contre ça valeur reste mutable indirectement.
